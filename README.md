# Student Management System

A comprehensive Student Management System developed in C++ using Object-Oriented Programming (OOP) principles. This application provides an efficient way to manage student records, including academic performance, attendance tracking, and report generation.

## Features

* Add new student records
* Search students by ID
* Update student information
* Delete student records
* Attendance management
* Grade calculation and report generation
* Sort students based on marks
* Persistent data storage using file handling
* Menu-driven user interface
* Object-Oriented Design with inheritance and encapsulation

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL (Vector, Algorithm)
* File Handling
* Inheritance
* Encapsulation

## Project Structure

The system is designed using a `Person` base class and a `Student` derived class, demonstrating inheritance and code reusability. Student data is stored in files, allowing records to persist between program executions.
